## 개발환경
1) 영화 상세 정보 제공
2) 로그인 & 회원가입
3) 영화 리뷰
4) 영화 및 간식 결제
5) 스낵바
<br><br>

- 개발언어 :  JAVA, TypeScript(Next.js), JavaScript, CSS <br>
- 사용툴: VSCode, IntelliJ<br>
- DB: MySQL <br>
- API : 영화진흥위원회 오픈 API<br>
- 요청 프로토콜 : REST<br>
- 응답 형식 : JSON<br>
